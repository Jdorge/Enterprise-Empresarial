# Contribuindo para Enterprise Empresarial

Obrigado por considerar contribuir para o Enterprise Empresarial! 🎉

## 🚀 Como Contribuir

### 1. Fork e Clone

```bash
# Fork via GitHub UI, depois:
git clone https://github.com/SEU_USUARIO/Enterprise-Empresarial.git
cd Enterprise-Empresarial
```

### 2. Criar Branch

```bash
git checkout -b feature/minha-contribuicao
```

**Convenção de Nomes:**
- `feature/` - Nova funcionalidade
- `fix/` - Correção de bug
- `docs/` - Documentação
- `refactor/` - Refatoração
- `test/` - Testes

### 3. Fazer Mudanças

- Escreva código limpo e documentado
- Siga os padrões do projeto
- Adicione testes quando aplicável

### 4. Commit

Use mensagens semânticas:

```
feat: adicionar integração com Slack
fix: corrigir timeout no MCP server
docs: atualizar guia de instalação
```

### 5. Push e Pull Request

```bash
git push origin feature/minha-contribuicao
```

Crie um Pull Request no GitHub com:
- Descrição clara do que foi feito
- Screenshots (se aplicável)
- Referência a issues relacionadas

## 📋 Checklist do PR

- [ ] Código testado localmente
- [ ] Documentação atualizada
- [ ] Sem credenciais hardcoded
- [ ] CI/CD passando
- [ ] Código revisado

## 🛡️ Segurança

**NUNCA** commite:
- API keys
- Senhas
- Tokens
- Dados sensíveis

Use sempre variáveis de ambiente.

## 📞 Dúvidas?

- Abra uma [Issue](https://github.com/Jdorge/Enterprise-Empresarial/issues)
- Entre em contato via email

Obrigado por contribuir! 🚀
